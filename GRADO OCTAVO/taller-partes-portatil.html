1. ¿Qué es un portátil?

Un portátil es un computador que puedes llevar a cualquier lado. Tiene todo integrado: pantalla, teclado, batería y componentes internos en un solo cuerpo compacto.

💡 Idea clave: El portátil es como un "computador de escritorio en miniatura". Sus componentes ocupan poco espacio y consumen poca energía.

2. Componentes internos

🧠 CPU (Procesador)

El cerebro. Procesa todas las instrucciones. Se instala con pasta térmica y se fija con un mecanismo de palanca.

📋 Placa Madre

La columna vertebral. Conecta todos los componentes. Se fija con tornillos al chasis.

📝 Memoria RAM

Memoria temporal. Se inserta con pinzas en los slots, presionando hasta que encaje.

💾 SSD / HDD

Almacenamiento. El SSD se inserta en ranura M.2 con destornillador. El HDD se fija con tornillos.

🎮 GPU

Tarjeta gráfica. Se fija con tornillos a la placa madre o chasis.

🔋 Batería

Fuente de energía. Se conecta con un cable a la placa madre y se fija con tornillos.

🌡️ Refrigeración

Ventiladores + heatpipes. Se instala con pasta térmica entre el CPU y el disipador, y se fija con tornillos.

📶 Wi-Fi

Tarjeta inalámbrica. Se inserta con pinzas en la ranura M.2 y se conectan las antenas.

3. Herramientas del técnico

  • 🔧 Destornillador de precisión: Para fijar componentes con tornillos (SSD, placa madre, GPU, batería).
  • 🤏 Pinzas: Para insertar componentes delicados como la RAM y la tarjeta Wi-Fi.
  • 🧊 Pasta térmica: Para mejorar la transferencia de calor entre el CPU y el disipador.
  • ⚡ Pulsera antiestática: Para proteger los componentes de descargas eléctricas.
📌 Importante: Cada herramienta se usa para componentes específicos. ¡Elegir la herramienta correcta es parte del trabajo del técnico!
